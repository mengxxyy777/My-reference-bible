# 最大公约数与同余原理

​	此处不涉及除法同余原理。

## 最大公约数

​	如何求两个数的最大公约数呢？

​	这里采用辗转相除法。更复杂的可能会在后续拓展。

### 辗转相除法

​	给出两个数字a和b（假设a > b），不断地用a去模b，将b的值给a，将a模b的值给b，直至b为0。

​	code非常简单：

```cpp
int gcd(int a, int b)
{
	  return b == 0 ? a : gcd(b, a % b);
}
```

### 最小公倍数

​	根据我们浅薄的数学知识可知最小公倍数的求法：b*gcd(a, b)。依然假设b是小的。

​	code：

```cpp
int lcm(int a, int b)
{
	  return a / gcd(a, b) * b;
}
```

## 同余原理

​	同余原理是处理取模的正确操作，在加或乘的时候，不能指望全加在一起之后在取模，这样必定会爆掉，所以才有了同余原理。

### 加法和乘法

​	同余原理的加法和乘法比较简单，((a+b)+(c+d)) % m等同于((a+b)%m+(c+d)%m) % m；

​	同样的，((a*b)(c*d)) % m等同于((a*b)%m*(c*d)%m) % m。乘法注意中间结果不要溢出。

### 减法

​	减法需要注意两数作差可能会为负数，这样我们加上一个模数就可以了。

​	(a-b+m) % m。
