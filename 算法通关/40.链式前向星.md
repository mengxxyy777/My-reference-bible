## 链式前向星

### 何为链式前向星？

​	链式前向星的本质其实是邻接表，只不过它存储的是图的边，对于每条边进行编号，然后通过next数组进行下一条边的跳转，通过head数组记录当前点的第一条边，to数组记录当前边要去哪，w数组记录当前边的权值。

​	我们发现，所有的数组都是固定大小的，head数组与点的个数有关，其余数组与边的数量有关，这让我们在空间要求苛刻的情况下也能快速解决问题。

### 链式前向星过程详解

​	我们知道，链式前向星是存储边的邻接表，需要存储边的信息，故我们可以这样写：

```cpp
int MAXN = 1000;
int MAXM = 10000;

int head[MAXN]; // 各个节点，每个节点后边跟着一个链表，链着的是与该顶点相邻的边 
int next[MAXM]; // 存放的是链表，每条边的下一条边在哪 
int to[MAXM]; // 存放的是每一条边的终点 
int weight[MAXM]; // 存放的是每一条边的权值 
int cnt = 1; // 记录当前边的编号
```

​	上述信息足以表示一张图了，且都是静态空间，对于空间要求严苛的情况下，也非常高效。

### 链式前向星边的存储

​	是一个头插法且插入边号的形式：

```cpp
void addedge(int u, int v, int w)
{
	next[cnt] = head[u];
	to[cnt] = v;
	weight[cnt] = w;
	head[u] = cnt++;
}
```

### 链式前向星边的遍历

​	对于每个顶点，既然链接了与之相邻的所有边，依次遍历即可。

```cpp
void through()
{
	for(int i = 1; i <= MAXN; i++)
	{
		for(int e = head[i]; e > 0; e = next[e])
		{
			//...
		}
	}
}
```
