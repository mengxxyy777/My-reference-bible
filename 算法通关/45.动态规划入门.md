# 动态规划入门

​	动态规划，运筹学的一个专业名词，我们解决算法类问题时，通常会遇到一些包含重复计算的问题，比如一些解空间树的展开，可能很多节点都是重复的。那么就准备一张表，把已经计算过的答案存在表里，等到下次再计算该问题求解时，直接从表中拿到答案即可。利用空间换取时间。

​	现在不谈一些什么无后效性的问题，为时尚早。先看一些经典的动态规划问题。

​	另外提一下，动态规划（dp）是非常庞大且难度较高的内容，通常需要大量的练习，同时需要对递归和搜索有足够的了解。动态规划的问题需要我们来构建思路，不像其他的算法有着固定的流程，动态规划本质是对递归的优化，而递归其实是尝试所有展开的过程。所以动态规划需要我们找规律，什么样的流程是正确的。不好把握。

## 一维动态规划

### 斐波那契数列

​	一个最经典、老生常谈的、每个人接触的第一个递归或动态规划的题目就是斐波那契数列。

​	何为斐波那契数列？从第三项开始，每项的值是前两项之和，依次递推下去。例如：1,1,2,3,5,8,11……

​	那么如何求该数列的第n项是多少呢？

```cpp
int fib(int i)
{
	if(i == 1)
	{
    	return 1;
	}
	if(i == 2)
	{
    	return 1;
	}
  
	return fib(i - 1) + fib(i - 2);
}
```

​	该代码就可以算出第一项、第二项为1时的斐波那契数列。

​	详细展开该递归过程，我们发现有很多状态是重复的。

## 二维动态规划

### 最长公共子序列问题

```cpp
for(int i = 1; i <= n; i++)
{
	for(int j = 1; j <= m; j++)
    {
    	if(s1[i-1] == s2[j-1])
        {
        	dp[i][j] = dp[i-1][j-1] + 1;
        }
        else
        {
        	dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
        }
	}
}
```

### 最长回文子序列

​	该问题也可以转化为最长公共子序列问题进行求解！

​	其实该问题属于区间DP的内容，但是难度不大。

```cpp
for(int i = 0; i < n; i++)
{
	for(int j = i; j < n; j++)
	{
		if(j == i)
		{
			dp[i][j] = 1;
		}
        else if(j == i+1)
        {
            dp[i][j] = (s[i] == s[j] ? 2 : 1);
        }
        else
        {
            dp[i][j] = 0;
        }
    }
}
for(int i = n - 3; i >= 0; i--)
{
    for(int j = i + 2; j < n; j++)
    {
        if(s[i] == s[j])
        {
            dp[i][j] = dp[i+1][j-1] + 2;
        }
        else
        {
            dp[i][j] = max(dp[i+1][j], dp[i][j-1]);
        }
    }
}
```
